## 🧠 Overview

A multi-turn chat assistant powered by GPT-4o vision. Upload any image — photo, chart, document, code screenshot — and ask questions about it in natural language. Maintains full conversation context across turns.

---

## ✨ Features

- ✅ GPT-4o / gpt-4o-mini vision API integration
- ✅ Image understanding: photos, charts, documents, diagrams
- ✅ Multi-turn conversational memory
- ✅ Quick-prompt buttons for common tasks
- ✅ Configurable system prompt, temperature, max tokens
- ✅ Demo mode works without API key

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/HaniaGhouse0407/Multimodal-AI-Assistant.git
cd Multimodal-AI-Assistant

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables (if needed)
cp .env.example .env
# Edit .env with your API keys

# 4. Run
streamlit run app.py
```
